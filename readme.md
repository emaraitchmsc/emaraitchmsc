## Projects

[**Royale Draft**](https://github.com/emaraitchmsc/royale-ban-draft)
Real-time multiplayer pick/ban tool for Clash Royale. Two players draft 8-card decks through alternating picks and bans. Built with Node.js, Socket.IO, and vanilla JS.

[**Google Meridian MMM**](https://github.com/emaraitchmsc/meridianmmm)
Marketing Mix Model using Google's Meridian library. Bayesian inference on daily marketing data (UK + US campaigns) to measure channel contributions and simulate budget reallocation. Python, Jupyter.

[**Direct Marketing Prediction**](https://github.com/emaraitchmsc/direct_marketing_prediction)
Predicting customer response to direct marketing campaigns. EDA, model training with cross-validation, threshold tuning, and feature analysis. Python, Jupyter.

[**Blood Type Prediction**](https://github.com/emaraitchmsc/bloodtype_prediction_r)
Supervised and unsupervised learning on a blood constituents dataset. PCA, bootstrap sampling. R.
